# Expense Tracker with Budget Planner

## Features
- Track income and expenses
- Set monthly budgets
- Overspending alerts
- Visual charts (Pie Chart)
- React + Node + MongoDB full-stack app

## Tech Stack
- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB
- Charts: Recharts

## How to Run
1. Start backend:
   cd backend
   npm run dev

2. Start frontend:
   cd frontend
   npm start
